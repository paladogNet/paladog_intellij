## ✔기능 사항

+ 팔라독은 좌우 컨트롤이 가능하다.
+ 팔라독은 "J"버튼을 통해 펀치스킬을 사용할 수 있다
+ 좀비(적)는 계속 소환이 되고, 쥐(아군)는 1번을 눌렀을 시 생산된다.
+ 보스(다크독)을 잡으면 게임이 끝난다.
</br>
